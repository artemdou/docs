## Überblick

In diesem Abschnitt wird der Bereich Checkout & Händler innerhalb der Einstellungen des Checkout-Demo-Shops erläutert.



Das JSON wird entsprechend den aktivierten oder deaktivierten Schaltern aktualisiert und kann kopiert werden.

## Händler

Nach Auswahl von *Händler & Checkout* im Menü __Einstellungen__ erscheint eine neue Seite, auf der Folgendes möglich ist:

- Wechsel der Checkout-Sprache
- Liste der aktiven Händler
- Möglichkeit, einen neuen Händler hinzuzufügen
- Wichtige Informationen über den Händler, wie zum Beispiel:
  - ID
  - Name
  - Nummer
  - CheckoutKey
  - Verfügbare Konten je nach Währung
