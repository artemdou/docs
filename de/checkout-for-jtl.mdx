## Bevor Sie beginnen

<Warning>
Bevor Sie beginnen, benötigen Sie ein [Checkout-Portal-Konto](https://portal.dibspayment.eu). Bitte beachten Sie die Anleitung:
- So [erstellen Sie ein Testkonto für Checkout](/create-a-checkout-portal-account),
</Warning>

## Übersicht

Unser JTL-Plugin ist die perfekte Erweiterung, mit der Sie das Potenzial von Checkout für Ihren JTL-Shop voll ausschöpfen können. Checkout unterstützt die meisten gängigen Zahlungsmethoden.

<Info>
Nachfolgend sehen Sie alle von Checkout angebotenen Zahlungsmethoden. Diese Liste bezieht sich auf alle Märkte.

Je nach Land oder Region kann die Liste variieren. Wenn Sie bei einer bestimmten Zahlungsmethode unsicher sind, ob diese in Ihrem Land oder Ihrer Region verfügbar ist, kontaktieren Sie bitte den [Support](https://developer.nexigroup.com/nexi-checkout/en-EU/support/) für weitere Informationen.
</Info>
