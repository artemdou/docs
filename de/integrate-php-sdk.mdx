## Überblick

Das Nexi Checkout PHP SDK ist eine leistungsstarke Bibliothek, die die Integration in die Zahlungsdienste von Nexi vereinfacht. Es bietet eine benutzerfreundliche Schnittstelle zum Erstellen von gehosteten und eingebetteten Zahlungslösungen, zur Verwaltung von Abonnements und zur Verarbeitung von Webhooks.

## Hauptmerkmale

- **Gehosteter Checkout**: Erstellen Sie gehostete Zahlungsseiten für nahtlose Transaktionen.
- **Eingebetteter Checkout**: Integrieren Sie das Zahlungs-iframe direkt in Ihre Anwendung.
- **Abonnementverwaltung**: Verwalten Sie Abonnements einfach.
- **Webhook-Verarbeitung**: Verarbeiten Sie Webhook-Ereignisse effizient.
