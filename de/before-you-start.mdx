Wie [hier](/checkout-paylink-and-one-page-shop-solution "Paylink-Voraussetzungen") beschrieben, sind die Voraussetzungen für die Einrichtung von Checkout Paylink folgende:

<Note>
1. Einen Paylink-Vertrag mit Checkout abschließen
2. Checkout Paylink-Konto erstellen
3. Ein Checkout-Konto besitzen
4. Einen Paylink-Instanznamen erstellen
</Note>

## 1. Paylink-Vertrag

Um Checkout Paylink zu nutzen, müssen Sie sich [hier](https://developer.nexigroup.com/nexi-checkout/en-EU/support/) an den Checkout-Vertrieb wenden. Sie erhalten per E-Mail einen Registrierungslink, über den Sie die Möglichkeit erhalten, ein Paylink-Konto zu erstellen.

Sobald Sie die Paylink-Kontoregistrierung abgeschlossen haben, müssen Sie den Paylink-Kontonamen mit dem Checkout-Vertrieb teilen. Schließlich unterstützt Sie der Checkout-Vertrieb bei den letzten Schritten zur Aktivierung und Einrichtung.

Sie müssen den Vertrag zur Aufnahme von Checkout Paylink in Ihren Checkout-Vertrag unterzeichnen.

Bei allen anderen Fragen, die nicht den Paylink-Vertrag betreffen, wenden Sie sich bitte [hier](https://developer.nexigroup.com/nexi-checkout/en-EU/support/) an den Support.

## 2. Checkout Paylink-Konto

Um Checkout Paylinks nutzen zu können, müssen Sie [ein Checkout Paylink-Konto erstellen](https://signup.nets-pay.link/en/email-validation/). Nachdem Sie den Checkout Paylink-Kontonamen an den Vertrieb gesendet und Ihr **Checkout Paylink-Konto** erstellt haben, definieren Sie anschließend Ihren **Paylink-Instanznamen**.

Eine ausführliche Anleitung zur [Erstellung Ihres Checkout Paylink-Kontos finden Sie hier](https://assets.ctfassets.net/di680may5c0k/1E1rqwjmUwJAmCIHkfXcmt/ac7bdf8affaf34550292e749d3abca07/Easy_Paylink_OneShop_Tutorials.pdf).

## 3. Checkout-Konto

Wenn Sie noch kein **Checkout-Konto** haben, können Sie [kostenlos eines erstellen](/create-nexi-checkout-portal-account).

Wenn Sie bereits ein bestehendes Checkout-Konto haben, kann dieser Schritt entfallen, da der Checkout Paylink-Vertrag vom Checkout-Vertrieb zu Ihrem Checkout-Vertrag und -Konto hinzugefügt wird.

## 4. Paylink-Instanzname

Mit der Einrichtung Ihres **Checkout Paylink-Kontos** definieren Sie Ihren **Paylink-Instanznamen**.

Der Paylink-Instanzname ist Ihr Checkout Paylink-Kontoname: Wenn Sie auf Ihre Checkout Paylink-Zahlungsseite über demo.nets-pay.link zugreifen, wäre der **Paylink-Instanzname** _demo_.
