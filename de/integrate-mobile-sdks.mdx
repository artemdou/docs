## Übersicht

Nets bietet zwei Möglichkeiten, Zahlungen auf Mobilgeräten zu akzeptieren: im mobilen Browser oder direkt in einer App über unsere Checkout Mobile In-App-SDKs (MiA).
Wählen Sie [iOS](https://github.com/DIBS-Payment-Services/Easy-IOS-SDK) oder [Android](https://github.com/DIBS-Payment-Services/Easy-Android-SDK), um zu beginnen.

<Columns cols={2}>
  <Card title="MiA – Checkout Android SDK" href="https://github.com/DIBS-Payment-Services/Easy-Android-SDK">
    Das MiA Checkout Android SDK ist eine Bibliothek, die die eingebettete Checkout-Integration in Ihrer Android-Anwendung ermöglicht.
  </Card>
  <Card title="MiA – Checkout iOS SDK" href="https://github.com/DIBS-Payment-Services/Easy-IOS-SDK">
    Das MiA – Checkout iOS SDK ist eine Bibliothek, die die Integration des gehosteten Easy Checkout in Ihrer iOS-Anwendung ermöglicht.
  </Card>
</Columns>
