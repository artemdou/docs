<section className="mx-auto mt-16 max-w-7xl px-4 sm:px-6 lg:px-8">
  <div className="max-w-3xl">
    <h2 className="text-3xl font-bold text-gray-900 dark:text-white">Schnellstart</h2>
    <p className="mt-3 text-base text-gray-600 dark:text-gray-400">
      In vier Schritten von null zu Ihrer ersten Testzahlung.
    </p>
  </div>

  <div className="mt-8">
    <Steps>
      <Step title="Checkout-Portal-Konto erstellen">
        Registrieren Sie sich für ein kostenloses [Checkout-Portal-Konto](/create-a-checkout-portal-account), um auf Ihre Sandbox zuzugreifen, Ihr Checkout zu konfigurieren und Zahlungen zu verwalten.
      </Step>
      <Step title="Integrationsschlüssel abrufen">
        Holen Sie sich Ihren **Checkout Key** (im Frontend verwendet) und Ihren **geheimen API-Schlüssel** (serverseitig verwendet) aus dem Portal. Siehe [Auf Ihre Integrationsschlüssel zugreifen](/access-your-integration-keys).

        <Note>
          Der geheime API-Schlüssel darf niemals öffentlich zugänglich gemacht werden. Nur der Checkout Key kann sicher in Ihrem Frontend eingebettet werden.
        </Note>
      </Step>
      <Step title="Eine Zahlung erstellen">
        Rufen Sie die Payment API von Ihrem Backend aus auf, um eine Checkout-Sitzung zu starten.

        ```bash cURL
        curl -X POST https://test.api.dibspayment.eu/v1/payments \
          -H "Authorization: $SECRET_API_KEY" \
          -H "Content-Type: application/json" \
          -d '{
            "checkout": {
              "integrationType": "EmbeddedCheckout",
              "url": "https://your-shop.example/checkout",
              "termsUrl": "https://your-shop.example/terms"
            },
            "order": {
              "currency": "EUR",
              "reference": "order-123",
              "items": [
                { "reference": "sku-1", "name": "T-shirt", "quantity": 1, "unit": "pcs", "unitPrice": 12900, "grossTotalAmount": 12900, "netTotalAmount": 12900, "taxAmount": 0, "taxRate": 0 }
              ],
              "amount": 12900
            }
          }'
        ```

        <Accordion title="Beispielantwort">
          ```json
          {
            "paymentId": "017a0000abcd1234ef567890",
            "hostedPaymentPageUrl": "https://test.checkout.dibspayment.eu/HostedPaymentPage/checkout?paymentid=017a0000abcd1234ef567890"
          }
          ```
        </Accordion>
      </Step>
      <Step title="Checkout in Ihre Seite einbetten">
        Verwenden Sie das [Checkout JS SDK](https://developer.nexigroup.com/nexi-checkout/en-EU/api/checkout-js-sdk), um das Checkout in Ihrer Seite zu rendern und auf das `payment-completed`-Ereignis zu reagieren.

        Setzen Sie mit dem vollständigen [Web-Integrationsleitfaden](/web-integration) für den End-to-End-Ablauf fort.
      </Step>
    </Steps>

  </div>
</section>
