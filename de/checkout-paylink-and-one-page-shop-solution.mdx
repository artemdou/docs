## Voraussetzungen

Bevor Sie mit Paylink beginnen, beachten Sie bitte, dass Sie für die Nutzung des Paylink-Dienstes in der Produktion Folgendes benötigen:

1. Schließen Sie eine Paylink-Vereinbarung mit Checkout ab, indem Sie sich für weitere Unterstützung an den Checkout Sales wenden
2. Erstellen Sie ein Checkout Paylink-Konto
3. Verfügen Sie über ein Checkout-Konto
4. Erstellen Sie einen Paylink-Instanznamen

## Checkout Paylink

__Checkout Paylink__ ermöglicht es Ihnen, Ihre Verbraucher zu einem sicheren, von Nexi Group gehosteten Zahlungsablauf zu leiten, indem Sie ihnen eine generierte URL senden. Die generierte URL ist der __Paylink__.

__Checkout Paylink unterstützt verschiedene Zahlungsarten:__

- __Einmalzahlungen__: Sofortige Belastung von Transaktionen (geeignet für konforme Produkte und/oder Dienstleistungen).
- __Reservierung__: Eine reservierte Transaktion muss belastet werden, damit Gelder überwiesen werden. Sofern zutreffend, wird empfohlen, reservierte Transaktionen innerhalb von 7 Tagen zu belasten, jedoch können verschiedene Zahlungsmethoden leichte Abweichungen aufweisen. Eine Übersicht finden Sie [hier](/manage-payments).

## Checkout One-Page-Shop-Lösung

Die __Checkout One-Page-Shop__-Lösung ermöglicht es Ihnen, Ihre Artikel auf einer Website mit einem integrierten, von Checkout gehosteten Zahlungsablauf zu präsentieren.

- Intuitive und einfache Anpassungsfunktionen, um Ihren Checkout One-Page-Shop entsprechend Ihren Markenanforderungen zu gestalten
- Die Seite ist responsiv für Desktop-, Tablet- und Smartphone-Anzeigen
- Mit Visa, Mastercard, Ratepay (Ratenzahlung und Lastschrift) und PayPal bieten wir die beliebtesten Zahlungsmethoden an
- Mit Checkout One-Page-Shop profitieren Sie von einem Komplettpaket mit einem intuitiven und einfachen Shopsystem, Hosting und Zahlungen – alles aus einer Hand.

__Es gibt drei Möglichkeiten, Ihren One-Page-Shop zu teilen:__

- __URL__: Teilen Sie einfach die URL des One-Page-Shops mit Ihren Verbrauchern
- __Modal-Fenster__: Wenn Sie bereits eine Website haben, können Sie eine Schaltfläche hinzufügen, damit der One-Page-Shop als Unterfenster vor dem Hauptfenster (Overlay) erscheint
- __iframe__: Sie können einfach einen Code-Schnipsel in den HTML-Code Ihrer Website integrieren, damit der One-Page-Shop in einem iframe erscheint

Sie können Ihren One-Page-Shop einfach in Ihrem Checkout Paylink-Konto erstellen und anpassen.
