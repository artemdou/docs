## Übersicht

Dieser Leitfaden enthält eine Beschreibung jeder Komponente innerhalb des Checkout Portal-Dashboards.

Die folgenden Komponenten finden Sie im Portal:

- Dashboard
- Berichte
- Unternehmen
- Add-on
- Suche

Sie können hier außerdem mehr über Zahlungen und Auszahlungen erfahren.

<Info>
Wenn Sie mit einem Testkonto beginnen, werden einige Funktionen erst angezeigt, nachdem Sie den Anmeldeprozess abgeschlossen haben, onboarded und live sind.
</Info>

## Dashboard

Das __Dashboard__ des Checkout Portal bietet Ihnen einen Überblick über wichtige Erkenntnisse zu Ihren Zahlungen und Bestellungen. Dazu gehören Daten zu:

- Neuen Zahlungen
- Gesamtumsatzvolumen
- Durchschnittlichem Bestellwert
- Auszahlungsbetrag
- Zahlungsmethoden
- Letzten Bestellungen

Diese wichtigen Erkenntnisse können so umgeschaltet werden, dass Daten zu einer bestimmten Währung und innerhalb eines bestimmten Zeitraums basierend auf der Ansicht eines bestimmten Tages, einer Woche oder eines Monats ein- oder ausgeschlossen werden.

## Berichte

Im Bereich __Berichte__ können Sie Auszahlungs- und Zahlungsberichte herunterladen.

<Info>
Nach 14 Tagen werden Berichte von der Berichtsseite entfernt.
</Info>

Die Berichtsübersicht zeigt:
- Erstellungsdatum
- Merchant ID
- Dokumentname
- Datumsbereich
- Währung
- Format

Für jeden generierten Bericht können Sie eine Reihe von *Schnellaktionen* ausführen.

![Berichte](https://images.ctfassets.net/di680may5c0k/1Sa1InsuXsWsMfSWrPYCAP/8ca4ffc25c61669240c57872574b16d8/ReportsView.png?w=2312&h=217&q=50&fm=png)

## Suche

Mit der Suchkomponente können Sie nach Zahlungen suchen, und zwar nach:

- Order ID
- Payment ID
- Name des Kunden
- E-Mail-Adresse des Kunden
- Telefonnummer des Kunden
- Subscription ID
- Kartennummer

Die Suchergebnisse können nach ihrem Status gefiltert und so umgeschaltet werden, dass Daten innerhalb eines bestimmten Zeitraums basierend auf der Ansicht eines bestimmten Tages, einer Woche oder eines Monats ein- oder ausgeschlossen werden.
