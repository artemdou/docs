## Overview

Nets offers two ways to accept payments on mobile devices. In the mobile browser or directly in an app using our Checkout Mobile in-App (MiA) SDKs.
Select [iOS](https://github.com/DIBS-Payment-Services/Easy-IOS-SDK) or [Android](https://github.com/DIBS-Payment-Services/Easy-Android-SDK) to get started.

<Columns cols={2}>
  <Card title="MiA - Checkout Android SDK" href="https://github.com/DIBS-Payment-Services/Easy-Android-SDK">
    MiA Checkout Android SDK is a library which facilitates the embedded Checkout integration in your Android application.
  </Card>
  <Card title="MiA - Checkout iOS SDK" href="https://github.com/DIBS-Payment-Services/Easy-IOS-SDK">
    MiA - Checkout iOS SDK is a library which facilitates the integration of the hosted Easy Checkout in your iOS application.
  </Card>
</Columns>
