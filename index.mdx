<section className="mx-auto mt-16 max-w-7xl px-4 sm:px-6 lg:px-8">
  <div className="max-w-3xl">
    <h2 className="text-3xl font-bold text-gray-900 dark:text-white">Quick Start</h2>
    <p className="mt-3 text-base text-gray-600 dark:text-gray-400">
      Go from zero to your first test payment in four steps.
    </p>
  </div>

  <div className="mt-8">
    <Steps>
      <Step title="Create a Checkout Portal account">
        Sign up for a free [Checkout Portal account](/create-a-checkout-portal-account) to access your sandbox, configure your checkout, and manage payments.
      </Step>
      <Step title="Get your integration keys">
        Grab your **checkout key** (used in the frontend) and **secret API key** (used server-side) from the portal. See [access your integration keys](/access-your-integration-keys).

        <Note>
          The secret API key must never be exposed to the public. Only the checkout key is safe to embed in your frontend.
        </Note>
      </Step>
      <Step title="Create a payment">
        Call the Payment API from your backend to start a checkout session.

        ```bash cURL
        curl -X POST https://test.api.dibspayment.eu/v1/payments \
          -H "Authorization: $SECRET_API_KEY" \
          -H "Content-Type: application/json" \
          -d '{
            "checkout": {
              "integrationType": "EmbeddedCheckout",
              "url": "https://your-shop.example/checkout",
              "termsUrl": "https://your-shop.example/terms"
            },
            "order": {
              "currency": "EUR",
              "reference": "order-123",
              "items": [
                { "reference": "sku-1", "name": "T-shirt", "quantity": 1, "unit": "pcs", "unitPrice": 12900, "grossTotalAmount": 12900, "netTotalAmount": 12900, "taxAmount": 0, "taxRate": 0 }
              ],
              "amount": 12900
            }
          }'
        ```

        <Accordion title="Example response">
          ```json
          {
            "paymentId": "017a0000abcd1234ef567890",
            "hostedPaymentPageUrl": "https://test.checkout.dibspayment.eu/HostedPaymentPage/checkout?paymentid=017a0000abcd1234ef567890"
          }
          ```
        </Accordion>
      </Step>
      <Step title="Embed the checkout on your page">
        Use the [Checkout JS SDK](https://developer.nexigroup.com/nexi-checkout/en-EU/api/checkout-js-sdk) to render the checkout in your page and listen for the `payment-completed` event.

        Continue with the full [web integration guide](/web-integration) for the end-to-end flow.
      </Step>
    </Steps>

  </div>
</section>
