## Overview

The Nexi Checkout PHP SDK is a powerful library designed to simplify integration with Nexi's payment services. It provides an easy-to-use interface for creating hosted and embedded payment solutions, managing subscriptions, and handling webhooks.

## Key Features

- **Hosted Checkout**: Create hosted payment pages for seamless transactions.
- **Embedded Checkout**: Integrate payment iframe directly into your application.
- **Subscription Management**: Manage subscriptions with ease.
- **Webhook Handling**: Process webhook events efficiently.

## Documentation

For detailed documentation, visit the [official Nexi Checkout PHP SDK repository](https://github.com/Nets-eCom/php-payment-sdk).
