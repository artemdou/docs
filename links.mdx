## Overview

In this section, the Links section inside of Settings of the Checkout demo store will be outlined.



## Webhooks

After selecting the *Webhooks* from the __Links__ menu, a new page will appear where it is possible to:

- Check the notifications based on the payment ID along with the:
  - Registered event
  - Timeframe of receival
  - Payload

You can filter and see all the webhook payment events which are also outlined in the API reference from [here](https://developer.nexigroup.comhttps://developer.nexigroup.com/nexi-checkout/en-EU/api/webhooks/).
